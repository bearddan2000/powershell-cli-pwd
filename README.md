# powershell-cli-pwd

## Description
A POC project to demonstrate
running Microsoft powershell
from linux.

## Tech stack
- powershell 7.1
- bash

## Docker stack
- docker-cli
- ubuntu:18.04

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
